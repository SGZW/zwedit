# zwedit
a simple online share code whiteboard (base on web)


## Technology Stack
* backend
    * code: java
    * httpserver: jetty
    * restful api: jersey
    * websocket: jetty-websocket
    * log: log4j
    * database: redis
* front end
    * code: es6
    * codeboard: codemirror.js
    * client-side mvc framework: vue.js
    * look: iview (base on vue.js)
    * http request: axios
    * websocket: raw websocket
